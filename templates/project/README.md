## Setup

Firstly the build tool `teapot` needs to be installed (which requires [Ruby][2]):
 
	$ gem install teapot

To fetch all dependencies, run:

	$ teapot fetch

## Usage

	$ teapot Run/$PROJECT_IDENTIFIER

	$ teapot Test/$PROJECT_IDENTIFIER
